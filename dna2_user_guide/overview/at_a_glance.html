DNA² at a Glance

DNA² is an BPM Suite

DNA² is a Rapid Application Development for people who build web applications focused on Processes. Its goal is to enable you to deploy application much faster than you could if you were writing code from scratch or using a framework, by providing a rich set of tools for commonly bussines problems, as well as a simple interface and logical structure to access these tools. DNA² lets you creatively focus on your project by minimizing the amount of time needed for a given Application.

As a Suite includes:

  • Unified Work Space with:
    • Task and InBox message panel
    • a User Interface
    • Supervision Panels
  • Execution Eviroment with:
    • Process Engine
    • Analysis Engine
    • Bussines Rules Engine
  • Process Modeler/Editor
  • Form Editor
  • User Layer with Groups,Permissions and Roles

DNA² is Free

DNA² is licensed under an Apache/BSD-style open source license so you can use it however you please. For more information please read the license agreement.

DNA² Runs on Web

DNA² is written over the exeptional Code Igniter Framework, and other Open Source tools that runs entirely on a compatible web Browser. Infrastructure:

  • Code Igniter 1.7
  • MongoDB
  • Jquery
  • Oryx

DNA² is Fast

The only thing you need is a webserver and a mongodb database server, both are really fast.

DNA² Uses M-V-C

DNA² uses the Model-View-Controller approach, which allows great separation between logic and presentation.

DNA² is Extensible

The system can be easily extended through the use of plugins, connectors and system hooks.[2do make links to plugins,connectors and system hooks]

DNA² includes a Template Engine

DNA² does come with a simple template parser that can be optionally used, to replace variables from tasks in your messages. ie.: your order no.: {order id} has been shipped

DNA² is Thoroughly Documented

Programmers love to code and hate to write documentation. We're no different, of course, but since documentation is as important as the code itself, we are committed to doing it. Our source code is well commented.

DNA² has a Friendly Community of Users

Our growing community of users can be seen actively participating in our Community Forums.
