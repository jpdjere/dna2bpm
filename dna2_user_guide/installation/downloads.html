Subversion Server

Subversion is a version control system.

Public subversion access is now available via http://dev.ellislab.com/svn/DNA²/trunk please note that while every effort is made to keep this codebase functional, we cannot guarantee the functionality of code taken from the repository.

Beginning with version 1.6.1, version snapshots (tags) are also available via the Subversion server via http://dev.ellislab.com/svn/DNA²/tags.
